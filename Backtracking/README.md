# Backtracking (DSA Problems)

This folder contains **Backtracking-based algorithms**, a powerful technique
used to solve problems by exploring all possible solutions and undoing choices
when a solution path fails.

Backtracking is commonly asked in **coding interviews** and **competitive programming**.

## ğŸ” What is Backtracking?
Backtracking is a recursive problem-solving technique where:
- We build a solution step by step
- If a step leads to an invalid solution, we **backtrack**
- Try another possibility
