# Binary Search

**Binary Search** is an efficient searching algorithm used on
**sorted arrays or lists**.

It works by repeatedly dividing the search space into half.

## Prerequisite
- The array must be **sorted**
- Random access should be possible (array / vector)

## How Binary Search Works
1. Find the middle element
2. If target == middle â†’ found
3. If target < middle â†’ search left half
4. If target > middle â†’ search right half
5. Repeat until found or search space becomes empty

## Types of Binary Search
- Iterative Binary Search
- Recursive Binary Search
- Binary Search on Answer
- Lower Bound / Upper Bound
- First & Last Occurrence

## Common Applications
- Searching in sorted arrays
- Finding boundaries
- Optimization problems
- Competitive programming problems

| Approach | Time | Space |
