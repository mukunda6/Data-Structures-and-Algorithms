# Recursion

**Recursion** is a programming technique where a function
calls itself to solve a problem.

## Key Concepts
- Base Case
- Recursive Call
- Call Stack
- Backtracking

## How Recursion Works
1. Function calls itself
2. Each call is stored in the call stack
3. Base case stops further calls
4. Stack unwinds and returns results

## Common Applications
- Factorial & Fibonacci
- Tree and Graph traversal
- Backtracking problems
- Divide and Conquer algorithms
- Dynamic Programming (top-down)
